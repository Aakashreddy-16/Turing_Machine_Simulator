Theory

Formal Definition

A Turing Machine is a 7-tuple:

M = (Q, Σ, Γ, δ, q₀, qₐ, qᵣ)

Components

  • Q — Finite set of states
  • Σ — Input alphabet (no blank)
  • Γ — Tape alphabet (Σ ⊆ Γ)
  • δ — Transition function
  • q₀ — Initial state
  • qₐ — Accept state
  • qᵣ — Reject state

Transition Function

δ: Q × Γ → Q × Γ × {L, R}

Given current state and tape symbol, the machine writes a symbol, moves left/right, and transitions to a new state.

Church-Turing Thesis

Any effectively computable function can be computed by a Turing Machine — it represents the ultimate model of computation.
